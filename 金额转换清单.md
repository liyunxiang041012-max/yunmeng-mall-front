# 金额转换完整清单

## 📋 转换规则
- **后端单位**: 分 (Long)
- **前端单位**: 元
- **转换公式**: 
  - 后端 → 前端: `元 = 分 / 100`
  - 前端 → 后端: `分 = Math.round(元 * 100)`

## ⚠️ 常见错误

### ❌ 错误示例
```javascript
// 错误1: 忘记转换
<span>¥{{ item.price }}</span>  // 显示 ¥5000 而不是 ¥50.00

// 错误2: 重复转换
item.price = item.price / 100 / 100  // 变成 0.5

// 错误3: 支付时忘记转回分
amount: payData.value.totalAmount  // 发送 100 而不是 10000

// 错误4: 使用整数除法
item.price = Math.floor(item.price / 100)  // 5099/100 = 50, 丢失小数
```

### ✅ 正确示例
```javascript
// 正确1: 转换为元,保留2位小数
item.price = parseFloat((item.price / 100).toFixed(2))

// 正确2: 支付时转换为分,四舍五入
amount: Math.round(payData.value.totalAmount * 100)

// 正确3: 显示时保留2位小数
<span>¥{{ item.price.toFixed(2) }}</span>
```
