## 二、数据库新增表

### 2.1 收藏表 `favorite`

```sql
CREATE TABLE `favorite` (
  `id`          bigint NOT NULL AUTO_INCREMENT,
  `user_id`     bigint NOT NULL COMMENT '用户ID',
  `item_id`     bigint NOT NULL COMMENT '商品ID',
  `create_time` datetime DEFAULT CURRENT_TIMESTAMP,
  PRIMARY KEY (`id`),
  UNIQUE KEY `uk_user_item` (`user_id`, `item_id`)

```

CREATE TABLE `shop_follow` (

  UNIQUE KEY `uk_user_shop` (`user_id`, `shop_id`)

## 三、接口详情

### 3.1 切换收藏状态

```
POST /it/favorites/toggle/{itemId}
```

| 项目 | 说明 |
|------|------|
| **路径参数** | `itemId` — 商品ID |
| **请求体** | 无 |
| **返回** | `Result<Boolean>` |

**返回示例：**

```json
{ "code": 200, "data": true, "msg": "ok" }
```

- `data = true` → 已收藏
- `data = false` → 已取消收藏

**业务逻辑**：同一接口 toggle，后端根据当前用户是否已收藏该商品决定收藏或取消。

---

### 3.2 检查收藏状态

```
GET /it/favorites/check/{itemId}
```

| 项目 | 说明 |
|------|------|
| **路径参数** | `itemId` — 商品ID |

**返回示例：**

### 3.3 获取店铺详情（核心接口）

```
GET /it/shop/{shopId}
```

| 项目 | 说明 |
|------|------|
| **路径参数** | `shopId` — 店铺ID |
| **返回** | `Result<ShopDetailVO>` |

#### ShopDetailVO 字段

| 字段 | 类型 | DB来源 | 说明 |
|------|------|--------|------|
| `id` | Long | `shop.id` | 店铺ID |
| `shopName` | String | `shop.shop_name` | 店铺名称 |
| `logo` | String | `shop.logo` | 店铺Logo URL |
| `description` | String | `shop.description` | 店铺描述 |
| `score` | String | **后端计算** | 综合评分，如 `"4.9"` |
| `goodRate` | String | **后端计算** | 好评率，如 `"98.6%"` |
| `avgShipTime` | String | **后端计算** | 平均发货时间，如 `"24h"` |
| `totalSales` | Number | **后端统计** | 总销量（整数） |
| `itemCount` | Number | **后端统计** | 在售商品总数 |
| `fansCount` | Number | **后端统计** | 粉丝数 |

#### 返回示例

```json
{
  "code": 200,
  "data": {
    "id": 1,
    "shopName": "云梦精选旗舰店",
    "logo": "/uploads/shop/logo_1.png",
    "description": "专注高品质数码产品，正品保障",
    "score": "4.9",
    "goodRate": "98.6%",
    "avgShipTime": "24h",
    "totalSales": 158000,
    "itemCount": 86,
    "fansCount": 32600
  }
}
```

> ⚠️ 前端销量展示做了格式化：`totalSales >= 10000` 时显示 `"15.8万+"`，后端直接返回原始数字即可。

### 3.4 获取店铺商品列表

```
GET /it/shop/{shopId}/items?sort={sort}&page={page}&size={size}
```

| 参数 | 位置 | 必填 | 说明 |
|------|------|------|------|
| `shopId` | Path | 是 | 店铺ID |
| `sort` | Query | 否 | `default`(综合) / `sales`(销量) / `price`(价格) / `newest`(最新) |
| `page` | Query | 否 | 页码，默认1 |
| `size` | Query | 否 | 每页条数，默认20 |

**返回**：`Result<Page<ItemVO>>`（标准分页结构）

#### ItemVO（商品卡片所需字段）

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | Long | 商品ID |
| `name` | String | 商品名称 |
| `mainImage` | String | 商品主图 |
| `price` | Number | 价格（**分**） |
| `salesCount` | Number | 销量（可选，显示"已售xx"） |

#### 分页返回示例

```json
{
  "code": 200,
  "data": {
    "records": [
      { "id": 1, "name": "极简降噪耳机", "mainImage": "/uploads/item/1.jpg", "price": 8900, "salesCount": 3200 },
      { "id": 2, "name": "轻奢皮革钱包", "mainImage": "/uploads/item/2.jpg", "price": 29900, "salesCount": 1800 }
    ],
    "total": 86,
    "size": 20,
    "current": 1,
    "pages": 5
  }
}
```
