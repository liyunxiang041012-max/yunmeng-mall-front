# 图书管理系统 — 开发规划文档

> 风格参考：云梦商城（暖金色调 · 大地色系 · 毛玻璃质感）
> 技术栈：SpringBoot 3.x + Vue 3 + Vite + Element Plus

---

## 一、项目概述

### 1.1 系统定位
一套轻量级图书管理系统，支持图书的增删改查、分类管理、借阅/归还记录等核心功能，界面设计借鉴云梦商城的高级质感。

### 1.2 核心功能模块

| 模块 | 功能点 |
|------|--------|
| **仪表盘** | 图书总数、在借数量、逾期统计、月度借阅趋势图 |
| **图书管理** | 图书列表、搜索/筛选、新增/编辑、上下架、批量导入 |
| **分类管理** | 分类树、新增/编辑/删除分类 |
| **借阅管理** | 借书登记、还书处理、借阅记录、逾期提醒 |
| **用户管理** | 读者列表、借阅卡管理、借阅历史 |

## 五、后端 API 设计

统一前缀：`/api`

### 5.1 认证模块
```
POST   /api/auth/login        登录
POST   /api/auth/register     注册（管理员专用）
```

### 5.2 图书管理
```
GET    /api/books             图书列表（分页 + 搜索 + 分类筛选）
GET    /api/books/{id}        图书详情
POST   /api/books             新增图书
PUT    /api/books/{id}        编辑图书
PATCH  /api/books/{id}/status 上下架
DELETE /api/books/{id}        删除图书
```

### 5.3 分类管理
```
GET    /api/categories        分类树
POST   /api/categories        新增分类
PUT    /api/categories/{id}   编辑分类
DELETE /api/categories/{id}   删除分类
```

### 5.4 借阅管理
```
GET    /api/borrows           借阅记录（分页 + 筛选）
POST   /api/borrows           借书登记
PATCH  /api/borrows/{id}/return  还书处理
GET    /api/borrows/overdue   逾期列表
```

### 5.5 读者管理
```
GET    /api/users             读者列表（分页）
POST   /api/users             新增读者
PUT    /api/users/{id}        编辑读者
PATCH  /api/users/{id}/status 启用/禁用
```

### 5.6 仪表盘
```
GET    /api/dashboard/stats   核心统计数据
GET    /api/dashboard/trend   月度借阅趋势
```

## 十、关键代码片段速查

### 10.1 Axios 封装（前端）

```js
// utils/request.js
import axios from 'axios'
import { ElMessage } from 'element-plus'

const service = axios.create({
  baseURL: '/api',
  timeout: 15000
})

service.interceptors.request.use(config => {
  const token = localStorage.getItem('token')
  if (token) config.headers.Authorization = `Bearer ${token}`
  return config
})

service.interceptors.response.use(
  res => {
    const { code, data, msg } = res.data
    if (code === 200) return data
    ElMessage.error(msg || '请求失败')
    return Promise.reject(new Error(msg))
  },
  err => {
    if (err.response?.status === 401) {
      localStorage.removeItem('token')
      window.location.href = '/login'
    }
    ElMessage.error(err.message)
    return Promise.reject(err)
  }
)

export default service
```

### 10.2 统一响应格式（后端）

```java
@Data
@NoArgsConstructor
@AllArgsConstructor
public class R<T> {
    private int code;
    private String msg;
    private T data;

    public static <T> R<T> ok(T data) {
        return new R<>(200, "success", data);
    }

    public static <T> R<T> fail(String msg) {
        return new R<>(500, msg, null);
    }
}
```
